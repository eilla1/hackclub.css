hackclub.css

hackclub.css is a classless CSS utility which applies the Hack Club theme styles to a given HTML page. Color mode (light or dark) is automatically switched based on the system mode.

How to use

Paste the following into the <head> of your HTML file.

<link rel="stylesheet" href="https://cloud-x52eok8sf-hack-club-bot.vercel.app/0hackclub.css">
